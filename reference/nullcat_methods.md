# Supported nullcat methods

Return the character vector of supported categorical randomization
methods.

## Usage

``` r
nullcat_methods()

nullcat_methods()
```

## Value

A character vector of method names.

Character vector of available method names.

## Examples

``` r
nullcat_methods()
#> [1] "curvecat" "swapcat"  "tswapcat" "r0cat"    "c0cat"   
```
